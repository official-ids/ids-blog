## 🤔 Что такое Roblox?

## 🛠️ Roblox Studio — обзор

Roblox Studio — это полноценная среда разработки игр. Вот что ты можешь делать:

### Интерфейс

| Панель | Описание |
|--------|----------|
| 🏗️ **Explorer** | Дерево объектов игры (Workspace, Lighting, Players и т.д.) |
| 📝 **Properties** | Свойства выбранного объекта |
| 🎨 **Toolbox** | Библиотека готовых моделей, скриптов, плагинов |
| 📐 **Terrain Editor** | Редактор ландшафта |
| 🔧 **Output** | Консоль для отладки скриптов |

### Основные объекты

```
Workspace
├── Baseplate          ← Земля/пол
├── Parts              ← 3D-объекты (блоки, сферы и т.д.)
├── Models             ← Готовые модели
└── Scripts            ← Lua-скрипты

ServerScriptService    ← Серверные скрипты
ReplicatedStorage      ← Общие ресурсы (клиент + сервер)
StarterPlayer          ← Стартовые скрипты игрока
StarterGui             ← Стартовый интерфейс
```

## 💻 Скриптинг на Luau

Luau — это диалект Lua, оптимизированный для Roblox. Разберём основы.

### Hello World

```lua
print("Hello, Roblox!")
```

### Создание частицы

```lua
local part = Instance.new("Part")
part.Parent = workspace
part.Size = Vector3.new(4, 1, 4)
part.Position = Vector3.new(0, 5, 0)
part.BrickColor = BrickColor.new("Bright blue")
part.Anchored = true
```

### Обработка событий

```lua
local part = workspace.Part

part.Touched:Connect(function(hit)
    local character = hit.Parent
    local humanoid = character:FindFirstChild("Humanoid")
    
    if humanoid then
        print(character.Name .. " коснулся детали!")
    end
end)
```

### Сервер vs Клиент

В Roblox важно понимать разницу:

```lua
-- Server Script (ServerScriptService)
-- Выполняется ТОЛЬКО на сервере
-- Безопасно для логики игры

local ReplicatedStorage = game:GetService("ReplicatedStorage")
local RemoteEvent = ReplicatedStorage:WaitForChild("MyEvent")

RemoteEvent.OnServerEvent:Connect(function(player, data)
    print(player.Name .. " отправил: " .. tostring(data))
end)
```

```lua
-- Local Script (StarterPlayerScripts)
-- Выполняется ТОЛЬКО у конкретного игрока
-- Для UI, ввода, эффектов

local ReplicatedStorage = game:GetService("ReplicatedStorage")
local RemoteEvent = ReplicatedStorage:WaitForChild("MyEvent")

RemoteEvent:FireServer("Привет сервер!")
```

## 🎨 Создание первой игры

### Obby (полоса препятствий)

Самый популярный тип игр для начинающих:

1. Создай **Baseplate**
2. Добавь **Parts** как платформы
3. Установи **Anchored = true** для каждой
4. Добавь **SpawnLocation** — точка возрождения
5. Добавь **Kill bricks** (детали, которые убивают):

```lua
-- Локальный скрипт в детали
script.Parent.Touched:Connect(function(hit)
    local character = hit.Parent
    local humanoid = character:FindFirstChild("Humanoid")
    if humanoid then
        humanoid.Health = 0
    end
end)
```

### Tycoon

Игры-тайкуны — ещё один популярный жанр:

- Игрок собирает деньги
- Покупает улучшения
- Автоматизирует производство

Основные компоненты:
- **Dropper** — генерирует деньги
- **Conveyor** — транспортирует объекты
- **Upgrader** — улучшает дропперы
- **Leaderstats** — отображает статистику

## 🎯 Советы для начинающих

### 1. Начни с малого

Не пытайся сразу создать MMORPG. Начни с простого obby или tycoon.

### 2. Изучи Lua

Lua — простой язык. Основи можно выучить за неделю.

```lua
-- Переменные
local имя = "Roblox"
local возраст = 18
local isActive = true

-- Функции
local function приветствие(имя)
    print("Привет, " .. имя .. "!")
end

приветствие("Мир")
```

### 3. Используй Toolbox

В Roblox Studio есть встроенный **Toolbox** с тысячами бесплатных моделей, скриптов и плагинов.

### 4. Присоединись к сообществу

- Discord-серверы разработчиков
- DevForum
- YouTube-туториалы

### 5. Публикуй и получай фидбек

Не бойся публиковать незаконченные игры. Фидбек от игроков — лучший способ улучшить проект.
