# Markdown — язык красивой документации ✍️

> *Markdown — это как писать текст, но он сразу выглядит красиво.*

---

## 📖 Что такое Markdown?

**Markdown** (`.md`) — лёгкий язык разметки. Ты пишешь обычный текст с парой символов — получаешь красиво отформатированный документ.

![Markdown](https://upload.wikimedia.org/wikipedia/commons/thumb/4/48/Markdown-mark.svg/320px-Markdown-mark.svg.png)

## 🎨 Базовый синтаксис

### Текст

| Синтаксис | Результат |
|-----------|-----------|
| `**жирный**` | **жирный** |
| `*курсив*` | *курсив* |
| `~~зачёркнутый~~` | ~~зачёркнутый~~ |
| `` `код` `` | `код` |

### Заголовки

```markdown
# H1 — Самый большой
## H2 — Подзаголовок
### H3 — Секция
#### H4 — Подсекция
```

# H1 — Самый большой
## H2 — Подзаголовок
### H3 — Секция
#### H4 — Подсекция

## 📊 Таблицы — продвинутый уровень

```markdown
| Фича | Простой | Продвинутый | Мастер |
|------|---------|-------------|--------|
| Текст | ✅ | ✅ | ✅ |
| Таблицы | ❌ | ✅ | ✅ |
| HTML | ❌ | ❌ | ✅ |
| Бейджи | ❌ | ❌ | ✅ |
```

## 💻 Блоки кода

### Python

```python
def hello_world():
    """Привет, мир на Python"""
    print("👋 Hello, World!")
    return True

if __name__ == "__main__":
    hello_world()
```

### JavaScript

```javascript
const hello = () => {
  console.log("👋 Hello, World!");
  return true;
};

hello();
```

### Bash

```bash
#!/bin/bash
echo "👋 Hello, World!"
git init && git add . && git commit -m "✨ Готово"
```
