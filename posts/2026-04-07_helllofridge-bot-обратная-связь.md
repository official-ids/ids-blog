# HellloFridge_bot — минималистичный бот обратной связи 🤖💬

> *Пользователь пишет → админ получает → админ отвечает. Вот и всё.*

---

## 🎯 Идея

Обычно боты техподдержки перегружены: категории, статусы, рейтинги, FSM-машины... 

А что если сделать **максимально просто**?

```
Пользователь → /start → пишет что угодно → админ получает
Админ → отвечает → пользователь получает ответ
```

Вот и весь бот.

---

## ⚡ Как это работает

### Для пользователя

```
👤: /start
🤖: Привет! Напиши мне что угодно — сообщение дойдёт до админа.

👤: У меня проблема с заказом #1234
🤖: ✅ Сообщение отправлено! Ждите ответ.

...через некоторое время...

🤖: 👨‍💼 Админ: Здравствуйте! Проблема решена, проверьте заказ.
```

### Для админа

```
📨 Новое сообщение от @username:
"У меня проблема с заказом #1234"

👨‍💼 Вы (ответ): Здравствуйте! Проблема решена, проверьте заказ.
```

---

## 🔧 Стек

- **Python 3.11+** + **aiogram 3.x**
- **SQLite** — хранение пользователей и сообщений
- **aiohttp** — для webhook (опционально)
- **Docker** — деплой в один команд

---

## 📋 Фичи

| Фича | Описание |
|------|----------|
| 📨 **Мгновенная доставка** | Сообщение пользователя → админу без задержек |
| 💬 **Двусторонняя связь** | Админ отвечает — пользователь получает |
| 📎 **Медиа поддержка** | Фото, файлы, голосовые — всё доходит |
| 🆔 **Идентификация** | Админ видит username и ID отправителя |
| 📝 **Логирование** | Вся история переписки в базе |
| 🚀 **Простота** | Минимум кода, максимум пользы |

## 🚀 Запуск

### Локально

```bash
# 1. Клонировать
git clone https://github.com/official-ids/HellloFridge_bot.git
cd HellloFridge_bot

# 2. Зависимости
pip install -r requirements.txt

# 3. Настроить .env
cp .env.example .env
# Вставить BOT_TOKEN и ADMIN_ID

# 4. Запустить
python main.py
```

### Docker

```bash
docker compose up -d
```

---

## ⚙️ Настройка

```env
# .env
BOT_TOKEN=123456:ABC-DEF...
ADMIN_ID=987654321
```

**Всё.** Две переменные — и бот работает.

Минимум файлов, максимум понимания.

## 🔒 Приватность

- Хранятся только **ID пользователей** и **текст сообщений**
- Нет сбора лишних данных
- Бот не хранит куки, IP, телефоны
