# Что такое GitHub — полный гайд для новичков 🐙

> *Если ты когда-нибудь слышал «запушь на гитхаб» и не понимал — этот пост для тебя.*

---

## 🤔 Что вообще такое GitHub?

**GitHub** — это платформа для хранения кода и совместной работы. Представь Google Docs, но для программистов.

![GitHub Logo](https://github.githubassets.com/images/modules/logos_page/GitHub-Mark.png)

| Фича | Что делает |
|------|------------|
| 📦 **Репозитории** | Папки с проектами и историей всех изменений |
| 🔀 **Ветки** | Параллельные версии кода (чтобы не сломать основное) |
| 🔃 **Pull Requests** | Предложение изменений с обсуждением |
| ⚡ **Actions** | Автоматизация: тесты, деплой, боты |
| 📄 **Pages** | Бесплатный хостинг сайтов прямо из репо |

---

## 🔑 Основные понятия

### Git vs GitHub

```
Git       = система контроля версий (программа на компе)
GitHub    = сайт, где хранятся твои Git-репозитории
```

### Репозиторий

Это твой проект со всей историей:

```
мой-проект/
├── src/          ← исходный код
├── README.md     ← главная страница
└── .git/         ← магия Git тут
```

### Коммиты

Каждый коммит — это «сейв» в игре:

![GitHub Chart](https://ghchart.rshah.org/1b65c9/official-ids)

## 🚀 Как начать за 5 минут

### Шаг 1: Регистрация

🔗 [github.com](https://github.com) → Sign Up → бесплатно

### Шаг 2: Первый репозиторий

```
New → Create Repository → Public → Create
```

### Шаг 3: Первый коммит

```bash
git clone https://github.com/ТВОЙ_НИК/мой-репо.git
cd мой-репо
echo "# Привет мир!" > README.md
git add .
git commit -m "🎉 Первый коммит"
git push
```

## 💡 Зачем это НУЖНО?

![Repos](https://img.shields.io/badge/Repos-10+-blue) ![Stars](https://img.shields.io/badge/Stars-100+-yellow) ![Contributions](https://img.shields.io/badge/Contributions-500+-green)

1. **Портфолио** — работодатель смотрит на твой GitHub
2. **Open Source** — тысячи бесплатных библиотек
3. **Коллаборация** — работай с кем угодно, откуда угодно
4. **Бэкап** — код в безопасности в облаке
5. **CI/CD** — автоматические тесты и деплой
