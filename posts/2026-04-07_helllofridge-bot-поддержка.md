# HellloFridge_bot — Telegram-бот техподдержки 🤖❄️

> *Бот, который решает все вопросы с поддержкой в одном окне Telegram.*

---

## 📌 Что умеет бот?

### 🛠️ Техподдержка

Пользователь пишет → оператор получает → проблема решена.

| Функция | Описание |
|---------|----------|
| 💬 **Обращения** | Создаёшь тикет — отвечаем в реальном времени |
| 📋 **Статус заявки** | Отслеживай прогресс прямо в чате |
| 🏷️ **Категории** | Проблема, вопрос, баг — сортировка по типу |
| ⏱️ **Время ответа** | Среднее время реакции — менее 10 минут |

### 💡 Предложения

Не нравится что-то? Есть идея? Пиши напрямую:

```
/предложить — отправь свою идею
```

Все предложения собираются и рассматриваются командой.

## 🎯 Кнопки и команды

| Команда | Что делает |
|---------|------------|
| `/start` | Запуск бота, главное меню |
| `/help` | Список команд |
| `/support` | Создать обращение в поддержку |
| `/suggest` | Отправить предложение |
| `/status` | Проверить статус заявки |
| `/faq` | Частые вопросы и ответы |

## 🔧 Как это работает

### Для пользователя

1. Открываешь бота → [@HellloFridge_bot](https://t.me/HellloFridge_bot)
2. Жмёшь **Начать** → выбираешь раздел
3. Пишешь вопрос → получаешь ответ

### Для оператора

```
Панель админа:
├── 📨 Входящие обращения
├── 📈 Статистика
├── 👥 Пользователи
├── 💡 Предложения
└── ⚙️ Настройки
```
